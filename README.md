## Fitness Workout App
A full-stack MERN fitness application designed to help users log, track, and manage workouts. It features authentication, CRUD functionality, and integration with external APIs for a richer fitness experience.

## Features
* **User Authentication:** Secure sign-in and sign-out using **Firebase Authentication**.
* **Workout Management (CRUD):**
  * **Create:** Add new workouts with details such as exercise name, muscle group, sets, reps, and weight (kg).
  * **Read:** View all saved workouts in an organized list.
  * **Update:** Edit existing workouts to adjust any details.
  * **Delete:** Remove workouts you no longer need.
* **Persistent Data:** Workout data is stored in **MongoDB** and synced with **local storage** for offline access.
* **YouTube Integration:** A dedicated page (via React Router) displays fitness-related videos fetched from the **YouTube API**.
* **Responsive Design:** Works seamlessly across desktop and mobile devices.
