# Fitness Workout App

A full-stack MERN fitness application designed to help users log, track, and manage workouts. It features JWT authentication, full CRUD functionality, analytics, and a clean, responsive UI.

## Features

### Backend Architecture
- **JWT Authentication** — Secure account creation and login with JSON Web Tokens.
- **RESTful API** — Organized with controllers, models, middleware, and routes following MVC conventions.
- **MongoDB Integration** — Persistent data storage using Mongoose schemas for exercises and users.

### Frontend
- **Authentication Flow** — Dedicated login page with account creation support.
- **Dashboard** — Displays exercise cards with options to create and edit entries via a modal form.
- **Search** — Filter exercises by muscle group using a search bar.
- **Sidebar Navigation** — View exercises by day or browse all at once.
- **CRUD Operations** — Full create, read, update, and delete support for workouts.
- **User Profile** — Update personal training details including goals, experience level, training preferences, and priority muscle groups.
- **Analytics Page** — Powered by Chart.js, visualizing total volume and total sets per muscle group, with a sidebar for per-muscle breakdowns and overall statistics.
